# 基于Java Web的教务管理系统 java833

## 项目概述

本项目是基于Java Web技术的教务管理系统。该系统遵循MVC三层体系结构，使用Struts2、Spring、Hibernate等框架，结合Tomcat服务器、JSP、CSS、HTML和JavaScript技术实现。数据存储采用MySQL数据库。

## 技术栈

- **后端：** Java、Struts2、Spring、Hibernate、MySQL
- **前端：** JSP、CSS、HTML、JavaScript

## 功能模块

### 学生服务子系统

- 查看个人信息
- 修改密码
- 申请修改学生信息
- 学生选课/查询已选课程
- 成绩查询
- 设计作品上传

### 教师服务子系统

- 查看个人信息
- 修改密码
- 申请修改教师信息
- 查看当前授课
- 录入成绩
- 添加课程
- 下载作品
- 统计成绩查询

### 管理员服务子系统

- 查看个人信息
- 学生管理
- 教师管理
- 管理员管理
- 审核请求
- 系统设置

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- **操作系统：** Windows/Linux均可
- **数据库：** MySQL
- **Web服务器：** Tomcat
